Руководство пользователя: "Терминал Охранника - Учет посещаемости"
Введение:
Добро пожаловать в "Терминал Охранника" - приложение, предназначенное для учета посещаемости и контроля парковки сотрудников. Ниже приведено руководство по использованию основных функций приложения.

1. Начало работы:
 1.1 Запуск приложения:
 -Откройте приложение "Терминал Охранника" на вашем устройстве.
 1.2 Интерфейс:
 -Вас встретит интуитивно понятный интерфейс, разделенный на различные разделы для ввода данных и отображения результатов.

2. Ввод данных:
 2.1 Номер пропуска:
 -Введите номер пропуска в соответствующее поле.
 2.2 Лимит часов (при регистрации гостя, при регистрации сотрудника поле нужно оставить пустым):
 -Если у пользователя есть лимит часов, укажите его в соответствующем поле.
 2.3 Имя и Фамилия:
 -Введите свои имя и фамилию.
 2.4 Действие:
 -Выберите одно из двух действий: "Вход" или "Выход".
 2.5 Парковка:
 -Если вам нужно зарегистрировать использование парковочного места, установите флажок.

3. Сохранение данных:
 3.1 Нажатие кнопки "Сохранить":
 -После ввода всех необходимых данных, нажмите кнопку "Сохранить" для регистрации посещения.
 3.2 Подтверждение сохранения:
 -После успешного сохранения, вы увидите подтверждение операции.

4. Просмотр данных:
 4.1 Таблица посещений:
 -В разделе "Таблица посещений" отображаются данные о всех посещениях, включая номер пропуска, имя, фамилию, время входа и выхода.
 4.2 Парковки:
 -Раздел "Парковки" показывает количество доступных парковочных мест для сотрудников и гостей.

5. Обновление данных:
 5.1 Обновление таблиц:
 -Обновление данных происходит автоматически при нажатии кнопки "сохранить".

6. Проверка лимита часов:
 6.1 Внимание к лимиту часов:
 -При выходе система проверяет, не превышен ли лимит часов для данного пользователя.
